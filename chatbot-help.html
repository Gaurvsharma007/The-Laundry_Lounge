FreshThreads Chatbot Help

The FreshThreads chatbot is now powered by Google's Gemini 2.0 Flash AI model through the Gemini API! This provides intelligent, conversational responses to customer inquiries about our laundry services.

Update: The chatbot now uses the Gemini API for AI-powered responses. The system includes a robust fallback mechanism that will activate automatically if there are any issues with the API connection.

How to Test the Chatbot

  1. Start the Node.js server:
    • Open a command prompt or PowerShell window
    • Navigate to your project directory
    • Run: npm run dev
  2. Open the laundary.html file in your browser:
    • Go to http://localhost:3001/laundary.html
    • Click on the chat button in the bottom right
  3. Start chatting! The AI-powered chatbot can help with:
    • Detailed information about services and pricing
    • Scheduling pickups with personalized recommendations
    • Order tracking assistance
    • Answering complex questions about laundry care
    • General inquiries about FreshThreads

Test Phrases

Try these example phrases to see the AI-powered chatbot in action:

  • "What's the best way to clean silk garments?"
  • "Can you explain your premium laundry service and how it differs from regular service?"
  • "I have a red wine stain on my white cotton shirt. What should I do?"
  • "What eco-friendly cleaning options do you offer?"
  • "My wedding dress needs cleaning. Do you offer special handling for delicate items?"
  • "Can I get an express service if I need my clothes back by tomorrow afternoon?"
  • "What loyalty programs or discounts do you offer for regular customers?"

Technical Details

The chatbot uses the following technologies:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js with Express
  • AI Integration: Google Gemini 2.0 Flash API
  • Real-time Updates: Socket.io
Note: The system includes a fallback mechanism that will provide pre-defined responses if the Gemini API is unavailable for any reason, ensuring the chatbot is always functional.

Troubleshooting

  • Chatbot not loading: Make sure your Node.js server is running and you're accessing the site through the correct port.
  • Slow responses: This could be due to high API traffic. The chatbot will show a typing indicator while waiting for responses.
  • Server not starting: Check console output for errors and make sure no other service is using port 3001.
